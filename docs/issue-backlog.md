# Issue Backlog

This repository includes a generated backlog of 300 issues for the StellarBounty project.

- JSON issue source: `issues/stellar_bounty_300_issues.json`
- GitHub issue creation script: `scripts/create_github_issues.sh`

## Issue categories

- Setup & DevOps
- Authentication & Users
- Wallet Integration
- Smart Contracts (Escrow + payouts)
- Backend API
- Frontend UI/UX
- Bounty & Submission Logic
- Notifications
- Reputation System
- Testing

## Usage

1. Ensure `gh` and `jq` are installed.
2. Authenticate with `gh auth login`.
3. Run `bash scripts/create_github_issues.sh` from the repository root.
